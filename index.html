About

Celebrity Meter

Celebrity Meter is a simple meter to calculate online popularity for a celebrity and compare each other celebrity. It collects data from Google, Wikipedia, IMDB and Twitter and calculates a rank value. For more info please visit How to section.

I developed this as my homework for Introduction to Natural Language Processing course of M.Sc. Programme in Computer Engineering in Yildiz Technical University. http://www.bologna.yildiz.edu.tr/index.php?r=course/view&id=5972&aid=3

How to

Usage

You can write a celebrity name each input box and press enter key, Celebrity Meter will collect data and calculate a rank value.
